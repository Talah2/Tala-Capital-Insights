# Portfolio Optimization Project

## Objective
Build an optimal investment portfolio using Modern Portfolio Theory and analyze risk-return tradeoff.

## Tools
- Python (pandas, numpy, matplotlib, yfinance)

## Methodology
- Collected stock data  
- Calculated returns and risk  
- Built covariance matrix  
- Generated Efficient Frontier  
- Calculated Sharpe Ratio  

## Key Insights
- Diversification reduces risk  
- There is an optimal portfolio for each risk level  
- Higher return = higher risk  

## Outcome
This project demonstrates data-driven investment decision making.
